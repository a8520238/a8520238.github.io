配置如下
1 安装node.js, 在官网安装.msi版本

2 使用npm install hexo命令安装 hexo， 之后的命令均为 npx hexo command

3 使用如下命令进行初始化

hexo init folder

cd folder

npm install

4 配置 _config.yml 更换主题

5 修改

deploy:

type: git

repository: https://github.com/a8520238/a8520238.github.io

branch: master

6 基本命令

npx hexo g 生成静态页面

npx hexo d 推送

npx hexo server 本地打开

npx hexo new test 新建文件
