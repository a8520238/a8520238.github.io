# 2 double
8.5对应的二进制 ——> 1000.1
 

计算符号位：因为为正数，所以符号位为 0。
 

计算指数位：规范化表示小数点左边只能有一位并且为1，所以为1.0001 * 2^3，规定指数位的底为2且幂加127（double类型加1023），所有指数位为130，转换位二进制——>1000 0010。
 

计算尾数位：规定尾数位去掉规范化前边的整数位的1，只保存小数，所以尾数位为——>0001

1bit（符号位） 11bits（指数位） 52bits（尾数位）
