indexOf和binarySearch的实现机制都是不同的

  • indexOf是基于equals来实现的只要equals返回TRUE就认为已经找到了相同的元素
  • 而binarySearch是基于compareTo方法的，当compareTo返回0 时就认为已经找到了该元素。

所以我们要保证这两者同步，compareTo相等时equals也相等，而compareTo不相等时equals也不相等
